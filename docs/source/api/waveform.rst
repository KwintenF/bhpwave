Waveform Module
===============

The complex waveform :math:`h = h_{+} - i h_{\times}` is calculated as a sum of many harmonics,

.. math:: h(t, r, \theta, \phi) = \frac{\mu}{r} \sum_{lm} H_{lm}(t) e^{i \Phi_{lm}(t)} {}_{-2} Y_{lm}(\theta, \phi)
